How It Works

Understanding the complete workflow from image upload to waste classification

Fig. 2. Flowchart for the working of System
1

Upload Input Image

Users begin by uploading an image of a waste item through the web interface. The system accepts common image formats (JPEG, PNG) and performs initial validation.

2

Select Classification Model

Users can choose from VGG16, InceptionV3, ResNet50, or the Ensemble approach. Each model has unique strengths, with the Ensemble providing the highest overall accuracy.

3

Image Processing

The system preprocesses the image based on the selected model's requirements. This includes resizing to 224×224 pixels, normalization, and model-specific transformations.

4

Classification Prediction

The processed image passes through the selected neural network model(s). For the Ensemble option, all three models process the image and a majority voting system determines the final classification.

5

Results Display

The system returns the classification result (Organic or Recyclable) along with a confidence percentage, helping users make informed waste disposal decisions.
